# ✨ Sparkly — Modern Educational Platform UI

**Sparkly** — bu zamonaviy dizayn, interaktiv elementlar va yuqori darajadagi foydalanuvchi interfeysiga (UI/UX) ega bo'lgan ta'lim hamda mentorlik platformasi loyihasi. Loyiha React muhitida, moslashtirilgan toza CSS va eng so'nggi komponentlar arxitekturasi asosida ishlab chiqilgan.

---

## 🚀 Asosiy Imkoniyatlar (Features)

* **Zamonaviy Landing Page:** Talabalar va mentorlarni birlashtiruvchi jozibador vizual dizayn.
* **Mukammal Ro'yxatdan o'tish (Sign Up) sahifasi:** To'liq validatsiya va xavfsiz forma boshqaruvi tizimi.
* **Jonli Bildirishnomalar:** `React Toastify` orqali foydalanuvchi amallariga mos (Success/Error) dinamik xabarlar.
* **Markazlashgan Stillar:** CSS `:root` o'zgaruvchilari orqali veb-sayt ranglar palitrasini oson boshqarish.
* **Professional Tipografika:** O'qishga qulay va zamonaviy **Inter** shrift tizimi integratsiyasi.
* **Vizual Effektlar:** Rasmlar uchun maxsus linear-gradient overlay va mukammal soyalar (`box-shadow`).

---

## 🛠 Texnologiyalar Tizimi (Tech Stack)

* **Freymvork/Kutubxona:** React.js (Vite)
* **Stillar:** Custom CSS3 (Dinamik `:root` o'zgaruvchilari bilan)
* **Shriftlar:** Inter Font Family
* **Xabarnomalar:** React Toastify
